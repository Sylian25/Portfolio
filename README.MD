# Sylian Aoudia — Portfolio Personnel

Portfolio personnel conçu et développé en HTML/CSS/JavaScript.

🌐 **Live** → [sylian-aoudia.vercel.app](https://sylianaoudia-portfolio.vercel.app/)

---

## Stack technique

- HTML5 sémantique
- CSS3 (variables, animations, responsive)
- JavaScript vanilla (IntersectionObserver, transitions, i18n)
- Fonts : Space Grotesk + Inter (Google Fonts)
- Déploiement : Vercel

## Fonctionnalités

- Loader animé
- Curseur custom
- Animation lettre par lettre sur le hero
- Scroll reveal sur toutes les sections
- Toggle dark / light mode (détection système automatique)
- Traduction FR / EN complète
- Formulaire de contact via Formspree
- Design responsive mobile / desktop
